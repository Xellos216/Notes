# Java Swing Calculator

## ✨ 개요
이 프로젝트는 기존에 제작한 CLI 기반의 산술 계산기를 **Java Swing 기반 GUI 계산기**로 확장한 버전입니다. 이 계산기는 **이항 연산과 단항 연산**을 모두 지원하며, 유지보수와 확장성을 고려해 **MVC 아키텍처**를 따릅니다.

---

## 🔊 주요 기능
- 기본 산술 연산: `+`, `-`, `*`, `/`, `^`
- 단항 연산: `sqrt`
- 소수점 입력 지원
- 모든 계산 결과를 히스토리로 표시
- 초기화(`C`) 및 계산(`=`) 기능
- 예외 처리 (예: 0으로 나누기 등)

---

## 📊 아키텍처 구조 (MVC 패턴)

| 레이어      | 클래스명               | 설명                                |
|------------|------------------------|-------------------------------------|
| 모델(Model) | `ArithmeticCalculator` | 실제 계산 로직 수행                 |
| 뷰(View)    | `CalculatorView`       | Java Swing을 이용한 GUI 구성        |
| 컨트롤러    | `CalculatorController` | 사용자 입력 처리 및 내부 상태 관리 |
| 실행부       | `CalculatorApp`        | 애플리케이션 실행 및 초기화         |

---

## 🛠 GUI 레이아웃 구조
```
+------------------------------------------+
|         [ 결과 출력 라벨(Label) ]         | ← resultLabel
+------------------------------------------+
|  [7] [8] [9]   [+]   [sqrt]              |
|  [4] [5] [6]   [-]   [^]                 |
|  [1] [2] [3]   [*]   [/]                 |
|  [0] [.] [C]   [=]                       |
+------------------------------------------+
|         [ 히스토리 TextArea / JList ]     |
+------------------------------------------+
```
- 입력: `JButton`
- 출력: `JLabel`
- 히스토리: `JTextArea` 또는 `JList`
- 전체 배치는 `JPanel`, `GridLayout`, `BorderLayout` 활용

---

## 📝 동작 흐름

1. **사용자가 버튼 클릭** (숫자 또는 연산자)
2. **Controller**가 해당 이벤트를 수신하고 상태 업데이트
3. `=` 입력 시:
    - Controller가 `ArithmeticCalculator.calculate()` 호출
    - 계산 결과는 View에 전달되어 출력됨
    - 결과는 히스토리에도 추가됨
4. `C` 입력 시:
    - 내부 상태와 화면 초기화

---

## 🌟 목표
Java Swing 기반으로 **실제 계산기와 유사한 사용자 경험**을 제공하면서, 구조적으로도 명확하고 확장 가능한 계산기를 제작하는 것이 목표입니다.
