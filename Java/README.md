# Java 학습 노트

이 레포지토리는 Java 기초부터 고급 문법까지의 학습 내용을 정리한 노트와 실습 프로젝트를 포함하고 있습니다.  
각 챕터는 주차별 강의 흐름에 맞춰 구성되어 있으며, 실습 중심의 프로젝트도 함께 포함되어 있습니다.

## 폴더 구조

```
Java/
├── Chapter01        # 변수, 입출력, 연산자 등 기초 문법
├── Chapter02        # 클래스, 인터페이스, 상속 등 객체지향 개념
├── Chapter03        # 예외 처리, 컬렉션, 람다, 스트림 등 고급 문법
└── Project
    ├── Calculator   # 콘솔/스윙 계산기 구현
    └── Kiosk        # 구조 주석 포함 키오스크 예제
```

## 구성 안내

- `Chapter01 ~ Chapter03`
  각 주차별 강의 내용을 정리한 학습 노트입니다.

- `Project/`  
  강의에서 배운 개념을 적용한 실습 프로젝트들이 포함되어 있습니다.

- 각 폴더 내부에는 해당 주제에 대한 `README.md`가 따로 정리되어 있습니다.
