# QA Summary - BOJ 9461 파도반 수열

## 📌 Q1. 이 문제도 수학적인 개념을 이해해야 풀 수 있는 문제 아닌가요?

아니요. 이 문제는 **복잡한 수학 공식**보다 **수열의 규칙**을 찾는 문제입니다.  
`P(n) = P(n-1) + P(n-5)`라는 **점화식을 유도**하고, 그걸 코드로 구현하면 됩니다.

---

## 📌 Q2. 그러니까 P(5)까지는 수열이 정해져 있고, 그 이후부터는 규칙이 생기니까 그걸로 함수 만들면 되는 거죠?

맞습니다.  
- `P(1)`부터 `P(5)`까지는 **초기값으로 고정**  
- `P(6)`부터는 **점화식** `P(n) = P(n-1) + P(n-5)`를 적용해서 계산합니다.  
이 규칙을 기반으로 `dp` 리스트를 반복문으로 채우면 됩니다.

---

## 📌 Q3. `dp = [0] * 101`이랑 `dp[1] = dp[2] = dp[3] = 1` 부분이 이해가 안 돼요.

- `[0] * 101`은 **길이 101짜리 리스트를 만들고**, 전부 0으로 초기화하는 코드입니다.
- `dp[1] = dp[2] = dp[3] = 1`은 **여러 인덱스에 같은 값을 한 줄로 할당**하는 문법입니다.
- 이 초기값 설정은 문제에서 주어진 `P(1)~P(5)` 값을 바탕으로 합니다.

## 📌 Q4. 예제 입력은 3줄인데 왜 출력은 2줄인가요?

첫 줄은 **테스트 케이스 개수 T**를 나타냅니다.  
그 다음 T줄은 각각 **N값 (테스트 케이스)**이고,  
출력은 **각 테스트 케이스의 정답만 출력**되기 때문에 **T줄만 출력**되는 게 맞습니다.

예시 입력:
```
2 -> T
6
12
```

출력:
```
3
16
```

## 📌 Q5. 리스트에서 `[0] * 101`에서 `*`는 왜 곱하기가 아니라 "공간을 만든다"는 의미인가요?

파이썬에서 `*`는 리스트에 대해 **반복 복제** 역할을 합니다.  
예를 들어:

```python
[0] * 3 → [0, 0, 0]
["a"] * 4 → ["a", "a", "a", "a"]
```

즉, [0] * 101은 0이라는 값을 101개 복사해서 리스트로 만든 것입니다.
