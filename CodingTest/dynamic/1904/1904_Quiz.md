### ❓ Q3. 다음 코드의 실행 결과는 무엇인가요? (입력: `n = 5`)

```python
n = 5
dp = [0] * (n + 2)
dp[1] = 1
dp[2] = 2
for i in range(3, n + 1):
    dp[i] = (dp[i-1] + dp[i-2]) % 15746
print(dp[n])
```

- A) 5  
- B) 8  
- C) 13  
- D) 7  

**정답: B (dp[5] = 8)**  
→ 사용자는 A(5)를 선택했으나, 정답은 8입니다.
