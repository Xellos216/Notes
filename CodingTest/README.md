사전 제공된 `코딩테스트 알고리즘 문제 100선`을 1문제씩 학습하는 방식으로 학습을 진행합니다.

### 📚 학습 방식

2. **풀이 흐름 암기 중심 학습**

3. **매일 실습과 복습 병행**
   → 오늘 문제 풀이 + 전날 문제 리마인드 방식으로 장기 기억 고정

4. **QA 및 의문점 정리**
