# 1904 - 01타일

## ✅ 문제 설명

지원이는 0과 1로 구성된 타일을 이용해 길이 N짜리 2진 수열을 만들고자 한다.  
단, 사용할 수 있는 타일은 `1` 또는 `00`뿐이며, 이를 이용해 만들 수 있는 수열의 수를 구하는 문제이다.

## ✅ 입력

- 자연수 N (1 ≤ N ≤ 1,000,000)

## ✅ 출력

- 만들 수 있는 모든 2진 수열의 개수를 15746으로 나눈 나머지 출력

## ✅ 예제

| 입력 | 출력 |
|------|------|
| 4    | 5    |

## ✅ 알고리즘 분류

- 동적 계획법 (DP)
- 점화식: `dp[n] = dp[n-1] + dp[n-2]`
- 초기값: `dp[1] = 1`, `dp[2] = 2`

## ✅ 풀이 흐름 요약

1. `dp` 배열을 n+2 크기로 초기화
2. 점화식을 이용해 3부터 n까지 dp 값을 채움
3. 각 단계마다 `% 15746` 연산 적용
4. 최종 결과는 `dp[n]` 값
