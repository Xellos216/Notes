### Q53: x만큼 간격이 있는 n개의 숫자 (Python3, Java)

**Python3**

```python
def solution(x, n):
    return [x * i for i in range(1, n + 1)]
```

**Java**

```java
public class Solution {
    public long[] solution(int x, int n) {
        long[] answer = new long[n];
        for (int i = 0; i < n; i++) {
            answer[i] = (long)x * (i + 1);
        }
        return answer;
    }
}
```

### Q54: 자연수 뒤집어 배열로 만들기 (Python3, Java)

**Python3**

```python
def solution(n):
    return [int(d) for d in str(n)][::-1]
```

**Java**

```java
public class Solution {
    public int[] solution(long n) {
        String s = String.valueOf(n);
        int[] answer = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            answer[i] = s.charAt(s.length() - 1 - i) - '0';
        }
        return answer;
    }
}
```

### Q55: `digit`은 정해진 키워드인가?

- 아니다. `digit`은 그냥 반복문에서 쓰는 변수 이름일 뿐이며 `x`, `ch`, `c` 등으로 바꿔도 된다.

### Q56: C에서 동적 메모리 할당을 해줘야 한다는 건?

- 정답 배열 크기를 모를 경우 `malloc()`을 통해 런타임에 메모리 확보 필요.
- 정적 배열은 크기 고정이라 입력값에 따라 변할 수 없기 때문.

### Q57: Kotlin에서 `digitToInt()`, `toIntArray()`가 안 되는 이유는?

- Kotlin 1.5 이상에서만 지원됨.
- 대체: `it - '0'`, `toTypedArray().toIntArray()`로 구현 가능.

### Q59: 제곱수 판별 (Python3, Java)

**Python3**

```python
def solution(n):
    x = n ** 0.5
    if x.is_integer():
        return int((x + 1) ** 2)
    return -1
```

**Java**

```java
public class Solution {
    public long solution(long n) {
        double x = Math.sqrt(n);
        if (x == (long)x) {
            return (long)Math.pow(x + 1, 2);
        }
        return -1;
    }
}
```

### Q60: `n ** 0.5`는 왜 제곱근인가?

- 수학적으로 `x ** 0.5 == sqrt(x)`이기 때문. (루트 연산)
- `math.sqrt(n)`도 동일한 의미

### Q61: `return`과 `answer =` 차이

- `return`: 값을 함수 바깥으로 **반환**함
- `answer =`: 변수에 값만 **저장**함 (출력 아님)
- `return`이 없으면 함수는 기본적으로 `None` 반환

### Q62: 숫자 자릿수를 큰 수부터 정렬 (Python3, Java)

**Python3**

```python
def solution(n):
    return int(''.join(sorted(str(n), reverse=True)))
```

**Java**

```java
import java.util.Arrays;

public class Solution {
    public long solution(long n) {
        char[] digits = String.valueOf(n).toCharArray();
        Arrays.sort(digits);
        StringBuilder sb = new StringBuilder(new String(digits));
        sb.reverse();
        return Long.parseLong(sb.toString());
    }
}
```

### Q63: Python에서 `True` 뜨는 이유?

- 마지막 표현식이 `bool` 타입일 경우 자동 출력됨 (대화형 인터프리터에서 주로 발생)
- `print()` 또는 `return`을 명시해야 원하는 결과가 출력됨

### Q64: Java 코드에서 `Array.sort()` 에러 이유는?

- 클래스 이름 오타: `Array ❌` → `Arrays ✅`
- `import java.util.Arrays;` 반드시 필요함
