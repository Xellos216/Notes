# 두 수의 합 구하기

## 🧠 문제 설명
정수 `num1`과 `num2`가 주어질 때, `num1 + num2`의 값을 반환하는 함수를 작성하시오.

> 예시  
> - 입력: `num1 = 2`, `num2 = 3` → 출력: `5`  
> - 입력: `num1 = 100`, `num2 = 2` → 출력: `102`  

---

## 🐍 Python 풀이

```python
def solution(num1, num2):
    answer = num1 + num2
    return answer
```

---

## ☕ Java 풀이

```java
class Solution {
    public int solution(int num1, int num2) {
        int answer = num1 + num2;
        return answer;
    }
}
```
