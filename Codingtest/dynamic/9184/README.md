## 백준 9184번 - 신나는 함수 실행

### 📌 문제 분류

- 유형: 동적 계획법 1 (재귀 + 메모이제이션)

---

### 📋 문제 설명 요약

재귀적으로 정의된 함수 `w(a, b, c)`를 구현하고, 다양한 입력에 대해 그 값을 출력하는 문제. 조건에 따라 함수가 여러 갈래로 분기되고 중복 호출이 발생하므로, \*\*메모이제이션(DP)\*\*을 적용해 최적화해야 함.

---

### ✅ 핵심 개념

- **재귀 함수**: 입력 값에 따라 스스로를 계속 호출하는 구조
- **메모이제이션**: 이미 계산한 결과를 저장해 중복 호출 방지
- **3차원 DP 배열**: `dp[a][b][c]`를 통해 `w(a, b, c)` 값을 저장
- **조건 분기**: 문제에서 제시한 조건에 따라 다른 재귀식 적용
- **재귀 깊이 제한 해제**: `sys.setrecursionlimit()` 사용 필요

---

### 💡 아이디어 요약

- `a, b, c <= 0`: return 1
- `a, b, c > 20`: w(20, 20, 20)로 처리
- `a < b < c`: 세 개의 다른 재귀 호출을 더하고 빼는 방식
- 나머지 경우: 네 개의 다른 재귀 호출을 조합해 계산

---

### 🧠 메모이제이션 구조

- `dp[a][b][c]`를 통해 한 번 계산된 결과를 저장
- a, b, c는 0\~20까지만 유효하므로 크기 21x21x21의 3차원 리스트 생성

---

### 📌 입력 및 출력 형식 요약

- 입력: `a b c` 정수 3개가 반복해서 주어짐 (종료: -1 -1 -1)
- 출력: 각 입력마다 `w(a, b, c) = 결과값` 형식으로 출력

---

### 🧪 예시

**입력:**

```
1 1 1
2 2 2
10 4 6
50 50 50
-1 -1 -1
```

**출력:**

```
w(1, 1, 1) = 2
w(2, 2, 2) = 4
w(10, 4, 6) = 523
w(50, 50, 50) = 1048576
```
