# Day 5 - 베르트랑 공준 (4948번) 문제 Summary

## 📌 문제 유형

- **기본 수학 2** / 소수 판별
- 에라토스테네스의 체 사용

## ✅ 핵심 아이디어

- 주어진 수 `n`에 대해 `(n < x ≤ 2n)` 범위의 소수 개수를 세는 문제
- 입력이 여러 줄이고, 마지막에 0이 들어오면 종료

| 핵심 처리 대상     | 설명                                       |
|------------------|------------------------------------------|
| 소수 판별 범위     | 2 ~ 246912 (2 × 123456)까지 미리 구해둠       |
| 소수 판별 방법     | 에라토스테네스의 체로 True/False 리스트 구성  |
| 종료 조건         | 입력값이 0이면 while 루프 종료               |

## 🔁 풀이 흐름 요약

1. 소수 판별을 위한 리스트 `is_prime`를 미리 생성하고 초기화
2. 에라토스테네스의 체로 소수 여부를 미리 계산
3. 입력을 한 줄씩 받아 `n < x ≤ 2n` 조건을 만족하는 소수 개수를 세고 출력
4. 입력값이 0이면 종료

## 💻 정답 코드 (Python)

```python
import sys

MAX = 123456 * 2
is_prime = [True] * (MAX + 1)
is_prime[0] = is_prime[1] = False

for i in range(2, int(MAX**0.5) + 1):
    if is_prime[i]:
        for j in range(i*i, MAX + 1, i):
            is_prime[j] = False

while True:
    n = int(sys.stdin.readline())
    if n == 0:
        break
    count = 0
    for i in range(n+1, 2*n+1):
        if is_prime[i]:
            count += 1
    print(count)
```

## 🎯 외워야 할 것

| 키워드                   | 설명                                      |
|--------------------------|-------------------------------------------|
| `에라토스테네스의 체`     | 소수 판별 알고리즘. 배수들을 제거해서 소수만 남김 |
| `is_prime = [True] * n` | True로 초기화 후 소수가 아닌 곳만 False 처리     |
| `while True + break`     | 입력 개수를 모를 때 루프 처리 방식               |

## 🔍 시간 복잡도

- 에라토스테네스의 체: O(N log log N)
- 각 입력마다 소수 개수 세는 부분: O(N)
- 전체적으로 매우 빠르게 동작함 (입력 크기 제한 내에서)
