# 📘 문제 요약: 백준 1002번 - 터렛

## ✅ 분류
- 기본 수학 2
- 기하학 (원의 방정식, 거리 계산)

## ✅ 문제 설명
두 터렛에서 각각 적과의 거리를 측정해 해당 적(류재명)의 위치가 될 수 있는 지점의 개수를 구하는 문제.

## ✅ 핵심 아이디어
두 원의 중심과 반지름이 주어졌을 때, 두 원의 교점 개수를 수학적으로 판단:
- **두 원이 일치**: 무한대 → 출력 `-1`
- **두 점에서 만남**: 출력 `2`
- **내접 or 외접 (접점 1개)**: 출력 `1`
- **서로 만나지 않음**: 출력 `0`

## ✅ 수학 개념 요약
- 중심 거리 제곱: `(x1 - x2)^2 + (y1 - y2)^2`
- 반지름 합의 제곱: `(r1 + r2)^2`
- 반지름 차의 제곱: `(r1 - r2)^2`

이 거리 제곱값들을 비교하여 경우를 나눔.

## ✅ 예시
```
입력:
3
0 0 13 40 0 37
0 0 3 0 7 4
1 1 1 1 1 5

출력:
2
1
0
```
