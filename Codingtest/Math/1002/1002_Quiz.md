### ✅ Q1. Python에서 `**` 연산자의 의미는 무엇인가요?

**사용자 답변:** 제곱을 의미  
**채점:** ✅ 정답  
**피드백:** Python에서 `**`는 거듭제곱 연산자입니다. 예: `2 ** 3 = 8`

---

### ✅ Q2. 두 원이 외접하는 경우, 중심 거리와 반지름의 관계를 수식으로 써보세요.  
(제곱 형태로 표현 가능)

**사용자 답변:** `d ** 2 = (x1 - x2) ** 2 - (y1 - y2) ** 2`  
**채점:** ❌ 오답  
**피드백:** 잘못된 수식입니다.  
- 올바른 조건: `(x1 - x2)^2 + (y1 - y2)^2 = (r1 + r2)^2`  
- 즉, `d² = (r1 + r2)²`이 외접 조건입니다.

### ✅ Q3. 다음 입력이 주어졌을 때, 출력값은 무엇인가요?
```
입력:
1
0 0 10 30 0 20
```

**사용자 답변:** 1  
**채점:** ✅ 정답  
**피드백:** 중심 거리 30, 반지름 합 30 → 외접 → 교점 1개 → 출력: `1`

---

### ✅ Q4. `elif`는 어떤 문법 구조의 줄임말이며, 어떤 상황에서 사용하나요?

**사용자 답변:** elseif, 조건문을 여러번 반복하고 싶을 때  
**채점:** ⚠️ 부분 정답  
**피드백:**  
- `elif`는 `else if`의 줄임말 → 맞음  
- 더 정확한 설명: 조건이 여러 개일 때, 첫 번째 조건이 아니라면 그다음 조건을 체크할 때 사용  
- 예:
  ```python
  if x > 10:
      ...
  elif x > 5:
      ...
  else:
      ...
  ```
