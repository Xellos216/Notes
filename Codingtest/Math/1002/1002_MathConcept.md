# 1002_MathConcept.md

## 📘 수학 개념 정리: 두 원의 교점 개수 분류

---

### ✅ 문제 본질

두 개의 원이 평면 상에서 어떤 위치에 있느냐에 따라 **서로 만날 수 있는 점의 개수(교점)**는 달라진다.  
이 문제는 해당 교점의 개수를 수학적으로 분류하는 문제이다.

---

### ✅ 주요 변수

- 두 중심 좌표: `(x1, y1)`, `(x2, y2)`
- 반지름: `r1`, `r2`
- 중심 거리 제곱:  
  	 `d² = (x1 - x2)² + (y1 - y2)²`
- 반지름 합 제곱:  
  	 `(r1 + r2)²`
- 반지름 차 제곱:  
  	 `(r1 - r2)²`

---

### 📊 경우의 수 분류

| 경우 | 조건 | 설명 | 교점 수 |
|------|------|------|--------|
| 같은 원 | `d² == 0 and r1 == r2` | 중심과 반지름 모두 같음 | -1 |
| 두 점에서 만남 | `r_diff² < d² < r_sum²` | 2개의 교점 | 2 |
| 외접 | `d² == r_sum²` | 바깥에서 접함 | 1 |
| 내접 | `d² == r_diff²` | 안에서 접함 | 1 |
| 서로 만나지 않음 | `d² > r_sum² or d² < r_diff²` | 너무 멀거나 안쪽에 있지만 닿지 않음 | 0 |

### ✅ 요약

- 단순 if문 문제가 아니다.
- 수학적으로 **두 원의 상대적 위치** 개념을 정확히 알아야 풀 수 있다.
- 특히 두 점에서 만나는 조건 `r_diff² < d² < r_sum²`은 반드시 암기할 것.
