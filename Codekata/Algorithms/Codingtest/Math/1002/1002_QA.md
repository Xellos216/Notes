### ✅ Q1. `**`는 제곱을 뜻하는 거 맞지?

**A1. 네, 맞습니다.**  
Python에서 `**`는 거듭제곱 연산자입니다.

- 예: `2 ** 3 = 8`
- 문제에서는 `(x1 - x2) ** 2`처럼 거리 제곱 계산에 사용됨.
- `math.pow()`보다 빠르고 직관적입니다.

### ✅ Q2. `elif`는 뭐야? `elseif`의 줄임말이야?

**A2. 네, `else if`의 줄임말입니다.**  
Python에서는 `elif`로 조건 분기를 이어가야 합니다. `elseif`는 문법 오류입니다.

- 구조 예시:
  ```python
  if 조건1:
      ...
  elif 조건2:
      ...
  else:
      ...
  ```

### ✅ Q3. 문법을 다 알아도 "두 원의 위치 관계 분류"를 모르면 못 푸는 문제 맞지?

**A3. 맞습니다.**  
이 문제는 기본 문법보다 **수학적 개념(기하학)**이 핵심입니다.

- 중심 거리, 반지름 합/차의 관계를 통해 경우의 수 분류
- 교점 개수를 수학적으로 정확히 이해해야 조건 분기 구현이 가능
- 기하학 개념을 모르면 문법만으로는 문제 해결이 어려움

### ✅ Q4. 첫째 줄에 테스트 케이스의 T가 주어진다. 이건 왜 필요한가?

**A4. 테스트 케이스가 몇 개 주어지는지를 나타내는 숫자입니다.**

- 예: 입력이 다음과 같을 때
  ```
  3
  0 0 13 40 0 37
  0 0 3 0 7 4
  1 1 1 1 1 5
  ```
  첫 줄의 `3`은 아래 입력 세트를 **총 3개** 읽으라는 뜻입니다.

- Python 코드에서는 다음처럼 반복합니다:
  ```python
  T = int(input())
  for _ in range(T):
      ...
  ```

- 이 숫자가 없다면, 입력을 몇 줄 받아야 하는지 알 수 없어 반복문을 구성할 수 없습니다.

**즉, 입력의 ‘반복 횟수’를 명시하는 구조적 장치입니다.**
