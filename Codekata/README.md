# 🧠 CodeKata 프로젝트 개요

이 프로젝트는 Python, C, Java 3개 언어로 동일한 **알고리즘 문제**를 병렬 풀이하며,  
**SQL 문제**도 함께 병행하여 기초 문법과 문제 해결력을 동시에 훈련하는 것을 목표로 한다.

---

## ✅ 운영 원칙

- **문제 유형**:  
  - 알고리즘 입문 / 조건문 / 수학 / 반복문 등  
  - SQL SELECT 중심 기초 문제

- **풀이 언어**:  
  - 🐍 Python: 간결한 표현과 로직 구성 훈련  
  - 🧱 C: 자료형과 메모리 구조 중심 사고 훈련  
  - ☕ Java: 클래스 기반 구조와 타입 체계 익힘  
  - 🐬 SQL: SELECT, WHERE, JOIN 등 실전 쿼리 작성

- **기록 방식**:  
  - 날짜별 디렉토리(`DayXX/`)로 분류  
  - 그 아래 문제별 하위 폴더(`문제영문제목/`) 생성  
  - `.md` 파일 + 각 언어 실행 스크린샷 포함 가능

---

## 🗂️ 디렉토리 구조 예시

```
CodeKata/

│           ├── Two_Numbers_Difference.md

```

## 📌 파일명 규칙

- 마크다운: `문제영문제목.md`
- 이미지 캡처: 언어명 기준 (`python3.png`, `C.png`, `JAVA.png` 등)
