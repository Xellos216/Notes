### ✅ Q1. 이진법을 꼭 알아야 이해할 수 있는 문제인가요?

**아니요, 이진법 개념 없이도 풀 수 있습니다.**

- 여기서 "2진 수열"이란 단순히 `0`과 `1`로 이루어진 문자열 조합을 뜻합니다.
- 예를 들어 N=4일 때 가능한 수열: `0000`, `0011`, `1001`, `1100`, `1111` 등.
- 이진법 숫자(예: 110 → 십진수 6)를 다루는 문제가 아니라, 단순한 타일 조합 문제입니다.

---

### ✅ Q2. 점화식이란 무엇인가요?

**점화식(Recurrence Relation)**은 어떤 수열의 다음 항을 이전 항들로 표현한 수학식입니다.

- 예시: 피보나치 수열 → `f(n) = f(n-1) + f(n-2)`
- 이번 문제에서는 → `dp[n] = dp[n-1] + dp[n-2]`
- 의미: 길이 n짜리 2진 수열은
  - 길이 n-1 뒤에 '1'을 붙인 경우
  - 길이 n-2 뒤에 '00'을 붙인 경우의 조합으로 만들 수 있다는 뜻

---

### ✅ Q3. 왜 dp 리스트를 n+2 크기로 만들었나요?

**dp[i-1] + dp[i-2]를 계산할 때, 인덱스 에러를 방지하기 위함입니다.**

- 예를 들어 `i = 2`일 때 → `dp[0]`까지 접근해야 하므로 최소 `dp[0]`, `dp[1]`, `dp[2]` 필요
- 또한 N이 작을 때도 (`N=1`, `N=2`) 일관된 코드 실행을 위해 dp[2]까지 확보 필요
- 따라서 `dp = [0] * (n + 2)`로 넉넉하게 리스트 생성
