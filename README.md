# 🗂️ Notes Repository

이 리포지토리는 다양한 개발 분야에 대한 개인 학습 기록을 통합 관리하기 위한 공간입니다.  
각 디렉토리는 주제별로 분류되어 있으며, 실습 코드, 정리 문서, Q&A, 요약 노트 등이 Markdown 형식으로 정리되어 있습니다.

---

## 📌 사용 목적

- **개인 학습 기록 통합 관리**
- **Git을 통한 버전 관리 및 이력 추적**
- **지식 재사용 및 복습 용이성 확보**
- **실전 개발 대비 문서화 훈련**

---

## 🛠️ 사용 기술 스택

- Markdown (`.md`) 문서 기반
- Git CLI 및 GitHub 원격 저장소
- Bash 스크립트 및 Linux CLI 명령어
- SQL (MySQL, SQLite 등)
- Java, Kotlin, Python3
- Spring Framework (Core, Web, DataJPA)
- HTML/CSS/JS (Frontend 과제 일부)

---

## 🔄 업데이트 및 유지 방침

- **주제별 디렉토리 → 내용 단위 정리**
- **실습 중심 + 정리 중심 Markdown 혼합 관리**
- **학습 흐름 변경 시 디렉토리 리팩토링 적극 수행**
- **각 서브 디렉토리에는 README.md 작성 권장**
