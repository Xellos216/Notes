# Programmers SQL 문제 풀이 모음

이 디렉토리는 프로그래머스 SQL 풀이를 정리한 공간입니다.
각 문제는 고유 번호와 문제명을 기준으로 별도 폴더에 정리되어 있습니다.

## 📌 정리 기준

- 각 폴더명은 문제 번호 + snake_case로 작성합니다.
- 각 문제 디렉토리에는 다음 내용이 포함됩니다:
  - `README.md`: 문제 설명, 핵심 쿼리, 사용 문법 요약, 주석 중심 해설

## 📚 정리 목적

- 문제 풀이 과정에서 등장하는 문법/함수를 정리하고, 이후 실무에 재활용할 수 있도록 문서화합니다.
- 문제 간 공통 패턴(예: `GROUP BY + HAVING`, `INNER JOIN`, `FLOOR`, `MONTH` 등)을 파악해 SQL 스킬을 체계화합니다.
- GitHub에 업로드된 정리 파일은 CLI 환경에서 바로 확인하며 복습할 수 있도록 구성합니다.

## 🔜 향후 계획

- 문제별 `문법 요약`, `실행 쿼리 비교`, `복습 퀴즈` 항목을 체계적으로 포함
- 자주 틀리는 문법 분류 후 오답 노트 작성

---

📝 꾸준한 정리와 반복을 통해 SQL을 단순 암기가 아닌 실무 도구로 체득해나가는 것이 목표입니다.
