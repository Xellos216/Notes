# CodeKata SQL - Lv02 문제 목록

> Lv02는 SQL SELECT 실습의 심화 단계로, JOIN, GROUP BY, 날짜 처리 등 실무형 쿼리 중심으로 구성되어 있습니다.  
> 아래 목록은 `CodeKata/SQL/Lv02/` 디렉토리의 문제별 폴더와 연결됩니다.

---

## ✅ 진행 방식

- 문제 출처: 프로그래머스 SQL SELECT 실습 Lv.2~3 난이도
- 주제: JOIN, 날짜 조건, GROUP BY, CASE, 집계 함수, 서브쿼리 등
- 제출 방식: 각 문제별 디렉토리 내 `.md` 문서 및 실행 캡처 저장
