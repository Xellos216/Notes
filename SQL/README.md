# SQL_Study

SQL 학습 내용 정리 (CLI + GUI)

## 📁 폴더 구성

- `CLI`: CLI 환경에서 직접 쿼리를 실행하며 진행한 SQL 실습
- `Basic`: 기초 개념
- `Algorithms`: Programmers 문제 풀이 및 유형 정리


## 🧠 목적

- CLI 실습을 통해 SQL 기초 쿼리 작성 능력 강화
- GUI 환경을 통해 시각적 SQL 사용 능력 및 DB 탐색 훈련
-

## 🧰 도구

- CLI: MySQL 8.0
- GUI: DBeaver Community + MySQL
- Programmers
