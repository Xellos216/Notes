## 🔹 `str(n)`
- 정수를 문자열로 변환 → 각 자릿수 분리 가능

## 🔹 map(func, iterable)
- 반복 가능한 요소에 함수를 하나씩 적용한 이터레이터 생성

## 🔹 list(iterable)
- 반복 가능한 객체를 리스트 형태로 변환

## 🔹 리스트 컴프리헨션
- `[int(d) for d in str(n)]`  
  → 문자열 각 자리의 숫자를 다시 정수로 변환하여 리스트 생성

## 🔹 슬라이싱 문법
- `sequence[start:stop:step]`
    start: 시작 인덱스 (포함)
    stop: 끝 인덱스 (불포함)
    step: 증가 단위 (음수이면 역방향)

## 🔹 [::-1]의 의미
- `[::-1]`
    start 생략 → 처음부터
    stop 생략 → 끝까지
    step = -1 → 뒤에서 앞으로 한 칸씩 이동
== 전체 시퀀스를 역순으로 배열해라
