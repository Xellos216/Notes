### 📄 QA01.md – Java 기초 문법 & 용어 설명 (Q1 \~ Q11)

#### Q1. `class`, `public`, `static`, `void`, `main(String[] args)` 각각 무슨 뜻이야?

* `class`: 자바에서 모든 코드는 클래스 단위로 작성돼. `class Calculator`는 계산기라는 설계도를 정의한 것.
* `public`: 외부에서 이 클래스를 사용할 수 있도록 공개 범위를 설정한 키워드.
* `static`: 객체 생성 없이 사용 가능한 클래스 소속 함수나 변수에 붙임. `main` 메서드는 프로그램 시작점이므로 반드시 `static`.
* `void`: 반환값이 없다는 뜻. `main`은 실행만 하고 결과를 반환하지 않으니 `void`.
* `main(String[] args)`: 자바 프로그램이 실행될 때 제일 먼저 호출되는 메서드. `args`는 커맨드라인에서 전달받는 인자.

#### Q2. `Scanner`는 뭐야?

* 사용자 입력을 받을 때 사용하는 클래스.
* 예: `Scanner scanner = new Scanner(System.in);`
  → 키보드로 입력받는 스캐너 생성.

---

#### Q3. `nextLine()`, `next()`, `nextInt()` 차이점은?

| 메서드          | 설명                  | 예시 입력                |
| ------------ | ------------------- | -------------------- |
| `next()`     | 한 단어만 입력 (공백 이전까지만) | `3 + 5` 입력 시 `3`만 읽음 |
| `nextLine()` | 한 줄 전체를 읽음          | `3 + 5` 입력 시 전체 읽음   |
| `nextInt()`  | 정수를 입력받음            | `3`만 입력해야 함          |

---

#### Q4. `System.out.print()` vs `System.out.println()` 차이는?

* `print()`는 줄바꿈 없이 출력.
* `println()`은 출력 후 줄바꿈이 포함됨.

---

#### Q5. `+`, `-`, `*`, `/` 연산자는 어떻게 써?

* 자바에서도 `+`, `-`, `*`, `/`로 기본 연산 가능.
* 예: `int result = a + b;`

---

#### Q6. `String`은 뭐야?

* 문자열 자료형. `"hello"` 같은 문자 덩어리를 저장할 수 있음.
* 예: `String name = "홍길동";`

---

#### Q7. `System.out.println("결과: " + result);`의 의미는?

* `"결과: "`는 문자열
* `result`는 숫자
* `+`를 사용하면 자동으로 문자열로 이어짐 → `결과: 8`

#### Q9. `try`, `catch`, `while(true)`, `break`, `continue`, `switch` 뜻이 뭔지 모르겠어

* `try`-`catch`: 예외 처리 구조 (오류 나더라도 프로그램 멈추지 않게 함)
* `while (true)`: 무한 반복
* `break`: 반복문 중단
* `continue`: 이번 반복 건너뛰고 다음 반복
* `switch`: `if` 대체 조건 분기문 (값이 어떤 case에 해당하는지)

---

#### Q10. `NumberFormatException e`는 그냥 아무 문자여도 돼?

* `NumberFormatException`: 숫자 형식이 잘못된 경우 발생하는 예외 클래스 이름 (필수).
* `e`: 예외 객체를 담는 변수 이름은 아무거나 써도 되지만 `e`가 관례.

---

#### Q11. `Integer`는 뭐야?

* `int`의 객체 버전. 래퍼 클래스라고 불림.
* 예: `Integer.parseInt("123")` → 문자열을 정수로 변환
