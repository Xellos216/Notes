# 🧪 Bootcamp Mini Projects

이 디렉토리는 백엔드 개발 부트캠프 과정 중 진행한 **미니 프로젝트**들을 정리한 공간입니다.  
각 프로젝트는 실습 목적에 맞춰 작게 나누어져 있으며, 단계별 개발 경험을 축적하기 위한 구조로 구성되어 있습니다.

---

## 📁 디렉토리 구조

- `Calculator/`  
  CLI 기반 계산기 프로젝트입니다.  
  버전별(기초 구현 → 클래스화 → 고급 문법 적용)로 디렉토리를 분리해 발전 과정을 기록했습니다.

- `Intro/`  
  부트캠프 팀 소개 웹페이지 프로젝트입니다.  
  Firebase 연동, 카드 슬라이드, 인증 기능 등 실습 기반으로 구현한 내용이 포함되어 있습니다.

- `version/`  
  Firebase 보안 규칙, SDK 설정, 버전 변경 로그 등 각 프로젝트에서 사용한 환경설정 및 참조 문서를 저장한 공간입니다.

---

## 📌 목적

- 부트캠프 기간 동안 진행한 실습 프로젝트를 체계적으로 보관
- 프로젝트별 구조화된 디렉토리로 관리 용이성 확보
- GitHub Pages, Firebase 등 외부 서비스와의 연동 실습 결과 기록
- 개인 포트폴리오 및 복습용 문서 기반 확보

---

## 📎 기타 정보

- 각 하위 폴더에 포함된 `README.md`는 해당 프로젝트의 개요 및 구현 내용을 설명합니다.
- 실습 과정 중 사용된 명령어, 주요 코드, 기능 설명은 각 프로젝트 내 `.md` 파일에 정리되어 있습니다.

---

> 📂 최종 목표는 실습 위주의 경험을 토대로 실전 대응 능력을 키우고, 개별 프로젝트를 발전시켜 포트폴리오로 활용하는 것입니다.
