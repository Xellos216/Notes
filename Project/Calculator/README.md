# Java Console Calculator Project

본 프로젝트는 Java 기반 콘솔 계산기를 개발하면서 점진적으로 다음 개념들을 학습하는 데 목적이 있습니다:

- 클래스 미사용 → 클래스 기반 구조화 → 제네릭 및 람다 활용까지 확장
- 사용자 입력을 통한 연산 기능 구현
- 컬렉션과 메서드, 객체지향 캡슐화 개념 도입

총 3단계로 구성되어 있으며, 각 단계는 `v1`, `v2`, `v3` 디렉토리에 분리되어 있습니다.

---

## 📦 v1: 클래스 없이 기본적인 연산 수행

**목표:**
- Scanner로 양의 정수 2개와 사칙연산 기호 입력받기
- `if`, `switch` 등을 활용해 연산 수행 및 결과 출력
- 나눗셈 등에서 0으로 나누는 오류 처리 포함
- 사용자가 `"exit"` 입력 시까지 무한 반복 가능

**주요 포인트:**
- 제어문 로직 구현 (`if`, `switch`)
- 반복문 종료 키워드 처리 (`while`, `"exit"` 입력 시 break)
- 기본 연산기 구현 흐름에 대한 이해

---

## 📦 v2: 클래스 기반 연산기 구조로 리팩토링

**목표:**
- `Calculator` 클래스 도입
  - 연산 수행 메서드
  - 결과 저장 컬렉션 필드
- `App` 클래스에서 `Calculator` 인스턴스 활용
- 필드에 직접 접근하지 않고 `Getter/Setter` 메서드를 통해 접근 (캡슐화)
- 결과 중 가장 오래된 값 제거 메서드 구현

**주요 포인트:**
- 객체지향 구조 설계
- 캡슐화(Encapsulation) 원칙 적용
- 컬렉션 자료구조(List 등) 활용
- 클래스 간 역할 분리 (`App` ↔ `Calculator`)

---

## 📦 v3: Enum, 제네릭, 람다 & 스트림을 활용한 계산기 고도화

**목표:**
- 사칙연산 기호를 `Enum`으로 처리하여 구조화
- 다양한 타입의 숫자 입력 처리 (제네릭 도입)
- `double`, `int` 등 다양한 타입의 연산 수행 가능
- 결과 중 Scanner로 입력받은 값과 가장 가까운 값을 출력하는 기능 추가
- 람다 & 스트림 API 활용

**주요 포인트:**
- 제네릭 클래스 및 메서드 정의
- Enum 타입을 연산자 정보 관리에 활용
- Lambda 및 Stream으로 데이터 필터링/조회
- Java 8 이상 문법을 실습으로 학습
