# 🧮 Java 계산기 프로젝트 (Lv3 - Enum, 제네릭, + 실전 CLI 확장)

이 프로젝트는 Java의 고급 문법을 활용하여 **계산기의 구조와 기능을 단계별로 확장**한 버전입니다.  
기존 Lv3(Class + Enum + Generic)에 더해, 실제 사용자 입력에 가까운 **실전 CLI 인터페이스(V3CLI)**를 구현했습니다.

---

## ✅ 구현 목표

### Lv3 (기초 확장 버전)

- `Enum`을 통해 연산자 기호(+ - * /)를 타입으로 구조화
- `제네릭`을 활용하여 `int`, `double` 등 다양한 숫자 타입 지원
- `람다`을 활용하여 저장된 결과 중 통계적 값(최댓값 등) 추출

### V3CLI (실전 CLI 버전)

- 실제 입력 흐름 기반의 CLI 계산기 (`Main.java` 중심 구조)
- 단항 연산(`sqrt`) 추가 지원
- 계산 결과 실시간 저장 및 히스토리 출력
- 결과 통계 (최댓값, 최솟값, 합계) 자동 계산
- 소수점 5자리 고정 포맷 출력 (`%.5f`)
