# SQL Study with MySQL & DBeaver

> 엑셀보다 쉽고 빠른 SQL 실습 정리 (CLI + GUI 병행)

이 저장소는 SQL의 기본부터 고급까지 실무 중심으로 학습한 내용을 정리한 공간입니다.  
MySQL CLI 환경을 기반으로 실습하며, DBeaver GUI를 병행하여 데이터 시각화 및 쿼리 결과를 확인했습니다.

---

## 학습 요약

### SQL 기초 쿼리
- SELECT, FROM, WHERE 등 기본 구조 학습
- 비교·논리 연산자, BETWEEN, LIKE 활용
- 에러 메시지 해석법 등 실습 기반 문제 해결

### 계산과 정렬
- SUM, AVG, COUNT 등 기본 집계 함수
- GROUP BY와 ORDER BY를 활용한 정렬
- 범주별 통계 계산 및 실무 응용

### 문자 가공과 조건 분기
- REPLACE, SUBSTRING, CONCAT 등 문자 처리
- 조건문(IF, CASE) 사용법과 실무 예제
- User Segmentation 및 에러 해결 전략

### 서브쿼리와 조인
- Subquery를 통한 다단계 연산 수행
- JOIN을 이용한 다중 테이블 데이터 병합
- 조건별 수수료 계산과 실전 데이터 분석

### 고급 SQL과 실무형 쿼리
- 이상치/결측치 처리 전략
- PIVOT 가공 및 Window Function (RANK, SUM)
- 날짜 조건과 포맷 함수로 실무 자동화 구현

---

## 💻 실습 환경
- CLI: MySQL 8.0 (로컬 환경)
- GUI: DBeaver Community Edition
