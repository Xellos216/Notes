# 🧠 프로젝트 개요

이 프로젝트는 Python, Java 2개 언어로 동일한 **알고리즘 문제**를 병렬 풀이하며,  
**SQL 문제**도 함께 병행하여 기초 문법과 문제 해결력을 동시에 훈련하는 것을 목표로 한다.

---

## ✅ 운영 원칙

- **문제 유형**:  
  - 알고리즘 입문 / 조건문 / 수학 / 반복문 등  

- **풀이 언어**:  
  - 🐍 Python: 간결한 표현과 로직 구성 훈련  
  - ☕ Java: 클래스 기반 구조와 타입 체계 익힘  

- **기록 방식**:  
  - 그 아래 문제별 하위 폴더(`문제영문제목/`) 생성  
  - `.md` 파일 + 각 언어 실행 스크린샷 포함 가능

---

## 📌 파일명 규칙

- 디렉토리: `문제영문제목`
- 마크다운: `문제영문제목.md`
- 이미지 캡처: `문제영문제목.png`

---

## 🎯 목표

- 언어 간 사고방식 차이 훈련
- 기초 문법을 실전 풀이로 반복 숙달
- 정리된 결과물을 포트폴리오 자료로 활용
