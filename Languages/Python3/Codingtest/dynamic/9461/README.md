# 9461 - 파도반 수열

## ✅ 문제 유형
- 동적 계획법 (Dynamic Programming)
- 점화식 수열 계산

---

## ✅ 문제 설명 요약

파도반 수열은 삼각형의 변의 길이에 따라 정의되는 수열로, 다음과 같은 규칙을 따릅니다:

- P(1) = 1  
- P(2) = 1  
- P(3) = 1  
- P(4) = 2  
- P(5) = 2  
- 이후부터는 다음 점화식을 따름:  
  **P(n) = P(n-1) + P(n-5)** (단, n ≥ 6)

N이 주어졌을 때, P(N)의 값을 구하는 문제입니다.  
여러 개의 테스트 케이스에 대해 처리해야 합니다.

---

## ✅ 풀이 아이디어

1. 수열의 규칙을 먼저 분석한 뒤, 점화식을 도출한다.
2. 최대 N = 100 이므로, 미리 `dp[1]`부터 `dp[100]`까지 계산해두면 각 테스트 케이스에 대해 O(1)로 답을 출력할 수 있다.
3. 초기값은 문제에 명시된 대로 하드코딩하여 설정한다.

---

## ✅ 점화식

- **P(n) = P(n-1) + P(n-5)** (단, n ≥ 6)
- 초기값:  
  - P(1) = 1  
  - P(2) = 1  
  - P(3) = 1  
  - P(4) = 2  
  - P(5) = 2

---

## ✅ 예시

입력:
```2
6
12
```

출력:
```3
16
```

설명:
- P(6) = P(5) + P(1) = 2 + 1 = 3
- P(12)는 위 점화식을 반복 적용하여 계산하면 16
