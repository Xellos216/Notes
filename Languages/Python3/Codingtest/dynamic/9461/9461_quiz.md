## 🔹 Q1.파도반 수열 P(N)의 점화식을 정확히 쓰세요. (단, N ≥ 6일 때)

✅ 정답:  
P(N) = P(N-1) + P(N-5)

## 🔹 Q4.파이썬에서 [0] * 5의 동작 방식은 무엇인가?

✅ 정답:
리스트 [0]을 5번 복제해서 새로운 리스트 [0, 0, 0, 0, 0]을 만든다.

❗ 포인트: *는 숫자에서는 곱셈이지만, 리스트에서는 "반복 복제"의 의미다.
