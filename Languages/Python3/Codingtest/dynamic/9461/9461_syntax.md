# Syntax Summary - 9461 파도반 수열

## 📌 1. 리스트 초기화: `[0] * 101`

```python
dp = [0] * 101
```

* 의미: 크기가 101인 리스트를 만들고, 모든 값을 0으로 초기화
* `*` 연산자는 리스트에서 "반복하여 복제"의 의미로 사용됨
* 0번 인덱스는 사용하지 않음 (`1`부터 `100`까지 사용할 목적)

---

## 📌 2. 리스트 여러 값 동시 할당

```python
dp[1] = dp[2] = dp[3] = 1
dp[4] = dp[5] = 2
```

* 하나의 값을 여러 변수나 리스트 인덱스에 동시에 할당할 수 있음
* 위 코드는 `dp[1]`, `dp[2]`, `dp[3]`를 모두 1로 설정

---

## 📌 3. 리스트 컴프리헨션을 이용한 입력 처리

```python
N_list = [int(input()) for _ in range(T)]
```

* 여러 줄 입력을 간결하게 리스트에 저장
* `T`번 만큼 `input()`을 받아 `int`로 변환하여 리스트로 만듦

---

## 📌 4. 반복문을 이용한 점화식 적용

```python
for i in range(6, 101):
    dp[i] = dp[i - 1] + dp[i - 5]
```

* `P(n) = P(n-1) + P(n-5)` 규칙을 코드로 구현
* 이미 채워진 `dp[1]` \~ `dp[5]`를 기반으로 다음 항들을 반복적으로 채움

---

## 📌 5. 테스트 케이스 순차 출력

```python
for N in N_list:
    print(dp[N])
```

* 입력받은 각 테스트 케이스에 대해 결과 출력
* `dp` 리스트를 미리 다 만들어뒀기 때문에 매우 빠르게 처리됨 (O(1) 조회)
