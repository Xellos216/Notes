# 📁 Algorithms - Python3 

이 디렉토리는 **Programers Python3 알고리즘 문제 풀이**를 정리한 공간입니다.  
문제는 난이도에 따라 `Lv01`부터 `Lv08`까지 구분되어 있으며, 각 폴더에는 해당 레벨의 문제들이 정리되어 있습니다.

---

## ✅ Lv01 - 기초 수학 & 문자열
- 산술 연산, 평균, 약수, 배수 등 기본 수학
- 문자열 길이, 변환, 추출 등 문자열 기초

## ✅ Lv02 - 수학 응용 & 기본 알고리즘
- 문자열 정렬, 조건에 따른 수학 연산
- 간단한 반복문, 조건문 활용 문제

## ✅ Lv03 - 조건문 & 문자열 로직 심화
- 조건 분기, 문자열 조작 복합
- 정렬, 필터링, 패턴 찾기 등

## ✅ Lv04 - 시뮬레이션 & 문자열 처리
- 문자열 파싱, 상태 추적, 빈도 분석
- 간단한 시뮬레이션

## ✅ Lv05 - 완전탐색 & 구현
- 재귀, 브루트포스 탐색
- 2차원 배열, 위치 기반 로직

## ✅ Lv06 - 스택/큐 & 정렬 알고리즘
- 우선순위 처리, 정렬 조건 지정
- 게임 시뮬레이션, 조건 필터링

## ✅ Lv07 - 수열 변형 & 시뮬레이션 고급
- 비트 연산, 조건 기반 수열 시뮬레이션
- 최적화 조건을 찾는 문제

## ✅ Lv08 - 숫자 기반 ID 문제 (백준형)
- 숫자 ID 기반 문제 (예: 1010, 1436 등)
- 주어진 수에 맞는 규칙 구현 문제
