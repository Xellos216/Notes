# 01_NumbersDifference

## 🧠 Problem
정수 `num1`과 `num2`가 주어질 때, `num1`에서 `num2`를 뺀 값을 반환하는 함수를 작성하시오.

### Example  
- Input: `num1 = 2`, `num2 = 3` → Output: `-1`
- Output: `num1 = 100`, `num2 = 2` → Output: `98` 

---

### 🐍 Python Reference Code

```python
def solution(num1, num2):
    return num1 - num2
```
