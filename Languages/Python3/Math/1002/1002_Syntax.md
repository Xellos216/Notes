# 1002_Syntax.md

## 📘 문제에서 사용된 주요 문법 정리 (백준 1002번: 터렛)

### 1. `for _ in range(T):`
- 여러 테스트 케이스를 처리하기 위한 반복문.
- `_`는 반복 변수의 이름이 필요 없을 때 사용.

### 2. `map(int, input().split())`
- 문자열을 공백 기준으로 나눈 뒤 `int`로 변환하여 여러 변수에 할당.
- 예: `x1, y1, r1, x2, y2, r2 = map(int, input().split())`

### 3. `**` (거듭제곱 연산자)
- `(x1 - x2) ** 2`: 두 값의 차를 제곱.
- 제곱 계산 시 `math.pow()`보다 `**`가 더 빠르고 직관적임.

### 4. 조건문 (`if`, `elif`, `else`)
- 두 원의 관계를 거리 제곱을 기준으로 나누어 판단.
- `or`를 이용해 두 조건 중 하나라도 만족하는 경우를 체크.

### 5. 두 원의 관계를 수학적으로 표현
- 중심 거리의 제곱: `(x1 - x2) ** 2 + (y1 - y2) ** 2`
- 반지름 합/차의 제곱과 비교하여 위치 판별.
