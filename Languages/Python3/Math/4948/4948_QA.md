## ❓ 문제 풀이 중 이해 질문 및 답변 정리

---

### Q1. `n = int(input())`을 써야 하는지 `sys.stdin.readline()`을 써야 하는지 어떻게 판단하지?

- **답변**: 입력이 **많이 반복될 경우** `input()`보다 `sys.stdin.readline()`이 훨씬 빠르기 때문에 사용함.
- 이 문제는 여러 줄의 입력을 받아야 하고, 최대값이 커서 속도 차이가 발생할 수 있으므로 `sys.stdin.readline()`을 선택한 것.

---

### Q2. `is_prime`은 그냥 네가 정한 변수명?

- **답변**: 네, 단순히 **소수 여부를 저장하는 리스트**라는 의미로 "is_prime"이라는 이름을 사용한 것.
- 다른 이름으로 바꿔도 무방하지만, 의미가 명확해서 관습적으로 많이 쓰임.

---

### Q3. 
```python
is_prime = [True] * (MAX + 1)
is_prime[0] = is_prime[1] = False
```
이건 0과 1은 제외해야 하니까 입력한 코드?

- **답변**: 맞아요! 0과 1은 수학적으로 소수가 아니기 때문에 직접 False로 지정해줌.
- `[True] * (MAX + 1)`로 전부 True로 초기화했으니, 0과 1만 예외 처리로 바꾼 것.

---

### Q4. 에라토스테네스의 체는 수학 공식이야, 아니면 프로그래밍 공식 같은 거야?

- **답변**: 수학적인 소수 판별 알고리즘으로 **기원은 고대 그리스 수학자 에라토스테네스**에게서 시작됨.
- 지금은 프로그래밍에서 소수를 빠르게 찾기 위해 널리 사용하는 **알고리즘 공식**처럼 사용됨.

---

### Q5. `while True` 구문은 왜 있어야 하지?

- **답변**: 이 문제는 입력 개수가 정해져 있지 않아서 **"0"이 들어올 때까지 계속 입력을 받아야 함**.
- 그래서 `while True:`로 무한 루프를 만든 뒤, `if n == 0: break`를 통해 종료 조건을 따로 설정한 것.

---

### Q6. 아래 코드에서 첫 번째 줄은 무슨 뜻인지 잘 모르겠어.
```python
is_prime = [True] * (MAX + 1)
is_prime[0] = is_prime[1] = False
```

- **답변**:

#### 🔹 한 줄씩 뜯어보자

1. `MAX + 1`은 뭐야?

- 파이썬에서 리스트는 **0부터 시작**하는데, 우리는 0부터 MAX까지의 숫자를 다 다뤄야 해요.
- 예를 들어 MAX가 10이면, 0~10까지 총 11개의 숫자가 필요하죠.
- 그래서 리스트 크기를 `MAX + 1`로 만들어야 합니다.

2. `[True] * (MAX + 1)`은 뭐야?

- 파이썬의 리스트 곱셈 문법이에요.
- 예를 들어 `[7] * 3` → `[7, 7, 7]`처럼, 같은 값을 여러 번 넣은 리스트가 만들어져요.
- `[True] * (MAX + 1)`은 True가 (MAX + 1)개 들어간 리스트를 만드는 코드예요.

3. 왜 전부 True로 시작해?

- 우리는 이 리스트를 `인덱스 = 숫자`, `값 = 소수 여부(True/False)`로 쓸 거예요.
- 처음엔 **모든 수가 소수(True)** 라고 가정하고 시작해요.
- 나중에 에라토스테네스의 체 알고리즘으로 **소수가 아닌 것들만 False로 바꿔줄** 거예요.

#### 🔹 예를 들어 볼게

```python
MAX = 10
is_prime = [True] * 11  # [True, True, True, True, True, True, True, True, True, True, True]
is_prime[0] = is_prime[1] = False
```

- 이제 is_prime[2]부터 is_prime[10]까지가 전부 "소수일 수 있음"을 뜻하고,
- 0과 1은 소수가 아니니까 바로 False로 바꾼 거예요.

결론:  
> 이 코드는 "0부터 MAX까지의 숫자들이 소수인지 아닌지 판단하기 위해,  
> 처음엔 전부 True로 시작해서 이후 알고리즘으로 False로 걸러내는 준비 작업"이에요.
