### ✅ 1. 리스트 초기화

```python
dp = [0] * (n + 2)
```

- `n+2` 크기의 리스트를 0으로 초기화
- 인덱스 접근 오류 방지 및 초기값 설정을 위한 공간 확보

---

### ✅ 2. 동적 계획법 (DP)

```python
dp[1] = 1
dp[2] = 2
for i in range(3, n + 1):
    dp[i] = (dp[i - 1] + dp[i - 2]) % 15746
```

- 점화식: `dp[i] = dp[i-1] + dp[i-2]`
- 모듈러 연산 `% 15746`은 숫자가 너무 커지는 것을 방지

---

### ✅ 3. 모듈러 연산

```python
% 15746
```

- 문제에서 요구하는 출력 조건: **모든 결과를 15746으로 나눈 나머지로 출력**
- 계산 중간마다 모듈러 적용하여 오버플로 방지

---
