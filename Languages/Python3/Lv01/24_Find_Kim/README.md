# 24_Find_Kim

## 🧠 Problem
문자열 배열 `seoul`이 주어졌을 때, `"Kim"`이라는 이름이 **몇 번째(index)** 위치에 있는지 찾아서
`"김서방은 x에 있다"`라는 형식의 문자열로 반환하는 함수를 작성하시오.

### Example
- Input: `["Jane", "Kim"]`
- Output: `"김서방은 1에 있다"`

---

## 🐍 Python Reference Code

```python
def solution(seoul):
    x = seoul.index("Kim")  # Kim이 위치한 인덱스를 찾음
    return f"김서방은 {x}에 있다"
```

---

### index() 함수
python
x = seoul.index("Kim")
- `seoul` 리스트 안에서 `"Kim"`이라는 문자열이 **처음 등장하는 위치(index)** 를 반환합니다.

### 🔹 f-string 포맷팅
python
return f"김서방은 {x}에 있다"

- 문자열에 변수 `x`를 쉽게 끼워 넣을 수 있는 파이썬 문법입니다.
- 예: `x = 1`이면 → `"김서방은 1에 있다"`

---

## ❓ Q & A

### Q1. 문제에서 `seoul` 배열에 이미 `"Kim"`이 들어 있다는 건 우리가 직접 확인 안 해도 되는 건가요?
- 네. 문제에서 `"Kim"`은 무조건 들어 있다고 했기 때문에 
  우리가 `"Kim"`이 있는지를 따로 확인할 필요 없이 `.index("Kim")`를 바로 써도 됩니다.

### Q2. 그럼 출력이 `"김서방은 1에 있다"`인 이유는?
- 리스트는 0부터 시작합니다.
- `["Jane", "Kim"]`에서:
  - `Jane`은 인덱스 0
  - `Kim`은 인덱스 1
- 따라서 `"김서방은 1에 있다"`가 정답입니다.

- `"Jane"`은 단지 `"Kim"`의 위치가 1이라는 걸 테스트하기 위한 샘플 데이터일 뿐이며, 
우리가 처리해야 할 대상은 `"Kim"`의 위치 하나뿐입니다.
