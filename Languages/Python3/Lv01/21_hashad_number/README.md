# 하샤드 수 - 비교 연산자 vs if문

## 🧠 문제 설명
양의 정수 `x`가 하샤드 수이려면 `x`의 자릿수의 합으로 `x`가 나누어져야 합니다.  
예를 들어 18의 자릿수 합은 1+8=9이고, 18은 9로 나누어떨어지므로 하샤드 수입니다.  
자연수 `x`를 입력받아 `x`가 하샤드 수인지 아닌지 검사하는 함수를 작성하시오.

---

## ✅ 풀이 방식 1: 비교 연산자만 사용한 간단 버전

```python
def solution(x):
    digit_sum = sum(int(d) for d in str(x))
    return x % digit_sum == 0
```

### 📌 설명
- `x % digit_sum == 0` 은 **비교 연산자**입니다.
- 이 식의 결과는 `True` 또는 `False` 중 하나의 **불린값(bool)** 입니다.
- 따라서 `if`문 없이도 `return`을 바로 사용할 수 있습니다.

---

## ✅ 풀이 방식 2: if문을 사용한 명시적 버전

```python
def solution(x):
    digit_sum = sum(int(d) for d in str(x))

    if x % digit_sum == 0:
        return True
    else:
        return False
```

### 📌 설명
- `if` 조건문을 통해 논리 흐름을 더 명확히 볼 수 있습니다.
- 초보자에게는 코드의 의미를 이해하는 데 도움이 됩니다.

---

## 🧠 비교 연산자 `==`의 역할

| 표현식 | 결과 |
|--------|------|
| `5 == 5` | `True` |
| `7 == 3` | `False` |
| `x % digit_sum == 0` | `True` 또는 `False` |

- `==` 연산자는 **두 값이 같은지를 비교**하고, 그 결과를 `True` 또는 `False`로 반환합니다.
- 이처럼 비교 연산자는 **항상 불린값을 반환**합니다.
