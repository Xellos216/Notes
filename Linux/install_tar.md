# 🛠️ Ubuntu에서 `.tar.gz` 압축 파일 기반 프로그램 설치 가이드

이 문서는 `.tar.gz` 형식으로 배포되는 리눅스 프로그램을 Ubuntu에서 수동 설치하는 일반적인 절차를 설명합니다. 대부분의 GUI 프로그램이 이 구조를 따릅니다.

---

## 📦 사전 조건

- 설치하려는 `.tar.gz` 또는 `.tar.xz` 파일을 미리 다운로드 받아야 합니다.
- 기본 위치는 보통 `~/Downloads`입니다.

---

## ✅ 설치 절차

### 1. 터미널 열기

```bash
Ctrl + Alt + T
```

---

### 2. 다운로드 폴더로 이동

```bash
cd ~/Downloads
```

---

### 3. 압축 해제

```bash
tar -xvzf 파일이름.tar.gz
# 또는
tar -xvf 파일이름.tar.xz
```

> 압축을 풀면 보통 실행 파일과 리소스가 포함된 디렉토리가 생성됩니다.

---

### 4. 시스템 경로로 이동 (선택 사항)

```bash
sudo mv [생성된 디렉토리명] /opt/프로그램명
```

> `/opt`는 수동 설치한 프로그램을 저장하는 데 일반적으로 사용되는 디렉토리입니다.

---

### 5. 실행 파일에 대한 심볼릭 링크 생성

```bash
sudo ln -s /opt/프로그램명/실행파일명 /usr/bin/별칭
```

> 예: `실행파일명`이 `MyApp`일 경우 `/usr/bin/myapp` 으로 연결 가능

---

### 6. 실행 테스트

```bash
별칭
```

---

## 🖥️ (선택) 애플리케이션 메뉴에 등록하기

```bash
nano ~/.local/share/applications/프로그램명.desktop
```

아래 형식대로 작성:

```ini
[Desktop Entry]
Name=프로그램명
Exec=/opt/프로그램명/실행파일명
Icon=/opt/프로그램명/아이콘.png
Type=Application
Categories=Utility;
```

> 저장 후 애플리케이션 메뉴에서 검색 가능

---

## 📌 참고 사항

- 이 설치 방식은 **자동 업데이트가 지원되지 않습니다.**
- 새 버전이 나오면 수동으로 새 `.tar.gz` 파일을 내려받아 덮어씌워야 합니다.
- 시스템 통합(런처 등록, 아이콘 등록 등)은 수동 설정이 필요합니다.
