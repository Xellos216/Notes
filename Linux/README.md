# 🐧 Linux 실전 학습 정리 노트

이 디렉토리는 **Ubuntu 및 Linux 데스크탑 환경**에서의 실전 경험을 기반으로, 다음 네 가지 영역을 중심으로 작성된 실용 문서들을 모아둔 공간입니다:

- GNOME 기반 사용자 인터페이스 커스터마이징 및 초기 설정
- 다양한 패키지 설치 방식 및 포맷 이해
- 명령어와 텍스트 편집기 사용을 포함한 CLI 실전 활용법
- 복구 및 수동 제거 실습 등 문제 해결 중심의 사례

> 실습 중심으로 축적된 내용을 분류·정리한 것이며, 단순한 명령어 나열이 아닌 **흐름 중심의 실전 가이드**로 구성되어 있습니다.
